# Example Dataset

This directory contains synthetic example data for testing and demonstrating the toolkit.
No API key is needed to explore — pre-computed outputs are included.

## Files

| File | Description |
|------|-------------|
| `politikas_100.csv` | 100 hand-crafted synthetic proposals covering 11 policy categories |
| `annotations_example.parquet` | Pre-scored multi-run annotations (13 runs x 100 items, synthetic) |
| `labels_clean.parquet` | Cleaned/aggregated labels produced by `cli.py clean` |
| `annotations_mock.parquet` | Minimal mock annotations (legacy, for basic testing) |
| `reference_outputs/` | Pre-computed pipeline reports (validation, dimensionality, insights) |
| `triangulation_demo/` | Synthetic audit model labels for cross-model triangulation demos |

## Quick exploration (no API key needed)

```bash
# Browse pre-computed reports
cat data/examples/reference_outputs/dimensionality_report.md
cat data/examples/reference_outputs/insights_report.md
cat data/examples/reference_outputs/validation_report.md

# Or re-run the pipeline yourself
python cli.py clean --input data/examples/annotations_example.parquet --output output/labels_clean.parquet
python cli.py dimensionality --labels output/labels_clean.parquet --output-dir output/ --skip-predictive
python cli.py insights --labels output/labels_clean.parquet --items data/examples/politikas_100.csv --output-dir output/

# Try cross-model triangulation with mock data
python cli.py triangulate \
    --reference data/examples/triangulation_demo/reference_labels.parquet \
    --audit data/examples/triangulation_demo/audit_gemini_mock.parquet \
    --output-dir output/
```

## Column Specification — `politikas_100.csv`

| Column | Type | Required | Description |
|--------|------|----------|-------------|
| `item_id` | string | Yes | Unique stable identifier (e.g. `ex_001`) |
| `text` | string | Yes | Proposal text (English, 20-200 words) |
| `category` | string | Yes | One of the 11 policy categories below |
| `source` | string | Yes | `"synthetic"` for manually crafted examples |

## Note on synthetic annotations

The annotations in `annotations_example.parquet` are **not** real LLM outputs. They are generated by `scripts/generate_example_labels.py` with category-aware score profiles designed to produce meaningful pipeline results. To regenerate:

```bash
python scripts/generate_example_labels.py
python cli.py clean --input data/examples/annotations_example.parquet --output data/examples/labels_clean.parquet
```
